## Управление

### Камера
- **W, A, S, D** — Перемещение камеры.
- **Колесико мыши (зажатие) / Shift + ЛКМ** — Свободное перетаскивание камеры.

### Взаимодействие
- **ЛКМ (Левая кнопка мыши)**:
    - Выбор поселенца.
    - Растягивание рамки для выбора нескольких поселенцев.
    - Использование активного приказа (рисование заданий на карте).
- **ПКМ (Правая кнопка мыши)** — Приказать выбранным поселенцам идти в указанную точку.
- **Space (Пробел)** — Снять выделение со всех объектов.

### Горячие клавиши (Приказы)
- **Z** — Режим архитектора (постройка стен).
- **X** — Режим деструкции (снос стен/заданий).
- **C** — Режим рубки (вырубка деревьев).
- **T** — Выбрать всех поселенцев на карте.
- **H** — Скрыть/показать интерфейс (UI).

## Механики игры
- **Ресурсы**: Серебро, Камень, Еда.
- **Потребности**: Следите за уровнем голода и отдыха ваших поселенцев. Вы можете принудительно отправить их поесть или поспать через панель инспекции.
- **Задания**: Поселенцы автоматически выполняют назначенные вами задания (стройка, рубка, добыча), если они не заняты перемещением.

## Функции для разработчиков
Эти функции предназначены для тестирования и отладки игры:

- **P** — Открыть панель управления временем (Debug Time). Позволяет быстро менять текущий час и минуту.
- **L** — Включить/выключить туман войны (Fog of War).
- **R (Удерживать 2 секунды)** — Полная регенерация мира. Прогресс отображается на кнопке "Regen World".
